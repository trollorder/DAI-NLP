Steps to run
Run Main.py
Key in the product you are looking for. Ensure that it is not too general.
Wait for 30minutes to and hour for the report to be generated.
You can retrieve the report at the following path "reports\{}\{} recommendation report.docx" where {} is a placeholder for your product name

Input: Product Name
Output: Report
Dependencies:
Openapi key
youtube api key in google key.py
Transformers, openai, pandas, pickle libraries.
